# nlm - NotebookLM CLI 📚✨

Google NotebookLMをターミナルから！ 🪄✨

## 🚀 準備
```bash
go install github.com/tmc/nlm/cmd/nlm@latest
nlm auth
```

## 💻 基本
- `nlm list`: 一覧
- `nlm add <ID> <source>`: 追加
- `nlm chat <ID>`: チャット

詳細は `nlm --help` 🐾✨
